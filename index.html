SK Cloud Z Service Platform

Enterprise-Graded Container Orchestration Platform

Public, Private IaaS 구성 및 Enterprise 수준의 Kubernetes Private Cluster를 제공하고,
Container기반 MSA 코칭프로그램 운영, 다양한 오픈소스 카탈로그 및 add-on 서비스를 통한
비즈니스에만 집중할수 있게 해주는 오픈소스(Kubernetes)기반 Fully Managed Cloud Platform 서비스

Core

Cloud Native


  • 퍼블릭 클라우드 및 온프레미스 환경에 최적인 Cloud Native 기반 개발환경을 단시간내에 구축 가능합니다.
  • Cloud Native 기술을 통해 응용 프로그램을 마이크로 서비스로 배포하고 모든 서비스를 컨테이너에 패키징할 수 있습니다.
  • Cloud Native 기술을 통해 좀더 빠르게 비즈니스 제품을 시장에 선보일수 있습니다.


Enterprise-Ready Platform


  • Kubernetes 기반의 안정적인 Public/Private Container Platform Service를 제공합니다.
  • Cloud 기반 서비스 개발/운영을 위한 Add-on Service를 제공합니다. (일부 서비스는 개발중에 있습니다.)
  • 다양한 Cloud Platform 영역의 Managed Service를 제공합니다.
