Cloud Native

OpenSource 기반기술을 보유한 딜리버리, 미래를 위한 dependency 없는
네이티브 클라우드 오퍼링 플랫폼, Cloud Native의 시작 SK가 함께합니다.

SK Cloud Platform

  • Cloud Native기반 오픈소스에 전문적인 인력으로 구성되어 있으며 오픈 소스 소프트웨어 스택을 사용하여 응용 프로그램을 마이크로 서비스로 배포하고 모든 서비스를 컨테이너에 패키징하며 이러한 컨테이너를 동적으로 조율하여 리소스 사용을 최적화하도록 합니다.
  • 제공되는 클라우드 고유의 기술을 통해 소프트웨어 개발자는 훌륭한 제품을 더 빨리 개발할 수 있습니다.
  • Technical Architecture / Application Architecture / MSA, 컨설팅, 클라우드 관련 솔루션을 제공하고 있음
  • 퍼블릭 클라우드 및 온프레미스 환경에 최적인 Cloud Native 구축을 하고 있고 레퍼런스가 많다.
  • 쿠버, 프로메테우스, 플루언트디, envoy, jaeger, nats, helm, ansible, terraform, serverless 까지 모든 커버리지...
  • 지속적으로 추가예정
