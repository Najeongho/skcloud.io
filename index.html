Building a CLOUD For Everything

인프라스트럭처에서부터 애플리케이션 개발 코칭까지,
여러분의 비즈니스 변화를 가속할 수 있게 도와주는 다양한 서비스를 제공합니다.
SK C&C가 제공하는 Fully Managed Cloud Platform 위에서 당신의 서비스에 날개를 다십시오.

SK Cloud Z Service Platform

Enterprise-Graded Container Orchestration Platform

Public, Private IaaS 구성 및 Enterprise 수준의 Kubernetes Private Cluster를 제공하고, Container기반 MSA 코칭프로그램 운영, 다양한 오픈소스 카탈로그 및 add-on 서비스를 통한 비즈니스에만 집중할수 있게 해주는 오픈소스(Kubernetes)기반 Fully Managed Cloud Platform 서비스

Core

Cloud Native


  • Cloud Native기반 오픈소스에 전문적인 인력으로 구성되어 있습니다.
    (Technical Architecture, Application Architecture, MSA Architecture, Platform Architecture)
  • Cloud Native 오픈 소스 소프트웨어 스택을 사용하여 응용 프로그램을 마이크로 서비스로 배포하고 모든 서비스를 컨테이너에 패키징할 수 있습니다.
  • 제공되는 Cloud Native 기술을 통해 소프트웨어 개발자는 훌륭한 제품을 더 빨리 개발할 수 있습니다.
  • 퍼블릭 클라우드 및 온프레미스 환경에 최적인 Cloud Native 기반 개발환경을 단시간내에 구축 가능합니다.


Enterprise-Ready Platform


  • Kubernetes 기반의 Public/Private Container Platform Service
  • Cloud 기반 서비스 개발/운영을 위한 Add-on Service
  • Platform 영역의 Managed Service

Features

Container Monitoring Service

Z Monitoring

사용자의 시스템/서비스가 사용하는 Cloud Resource들의 상태와 다양한 정보를 모니터링하기 위해 필요한 Monitoring용 System을 서비스로 제공합니다.

Container Logging Service

Z Log

다양한 System, Platform, Application, Service의 모든 로그를 통합 관리하는 Storage와 검색 기능을 제공합니다.

Multi-Cloud Continuous Integration and Continuous Deployment Service

Z Pipeline

대부분의 퍼블릭 클라우드를 지원하며, Vmware, OpenStack 과 프라이빗 클라우드 또는 Kubernetes 플랫폼을 동시에 지원하는 CI/CD 서비스입니다.

START READING

Dedicated Managed DBaaS Platform

Z DB

최신 오픈 소스 데이터베이스를 기반으로 하는 Dedicated Managed DBaaS 플랫폼으로 Enterprise 기업에 적합한 애플리케이션 개발에 사용할 수 있습니다.

START READING

Serverless(FaaS) Service

Z Action

별도의 서버 없이 함수 애플리케이션을 빠르게 만들어서 실행할 수 있게 해주는 서비스. 서버 관리를 할 필요가 없으며, 애플리케이션이 실행되는 만큼의 비용만 지불하면 됩니다.

Multi-Cloud Infrastructure Automation

Z Composer
